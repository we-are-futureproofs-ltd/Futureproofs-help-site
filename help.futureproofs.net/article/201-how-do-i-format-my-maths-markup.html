How do I format my Maths markup?

Once you have entered Maths mode, there are a wide range of possibilities for your Maths. This page contains a comprehensive list of the options and how to enter them.
The basic principle is that you can type everything from your keyboard. However, we also provide a toolbar for common operations to make basic Maths simpler to type.
If it is possible to type a character from the keyboard (like plus or equals) then you can just do so. Other symbols can be entered using a condensed form of their standard names, prefixed with a \ character - your typing will be replaced by the specific formatting you have requested.
You end a command using the space bar or tab key (shown in this guide as [SPACE] or [TAB]). For example, if you've entered an exponent and want to continue with your expression, you might type: x^2 [TAB] + 3 = 12.
You can also enter commands after highlighting some text, and it will be applied to your selection. So, for example, typing 3 / [TAB] 4 will give the same result as typing 3, then selecting that character with keyboard or mouse and then typing / 4. This can be useful for building up complex expressions.
Finally, you can type normal text as well as Maths, using the Text button on the Formatting toolbar. In this mode, you can type words as normal (but without formatting), so that you can enter Maths expressions that are part of continuous text as well as displayed equations.
